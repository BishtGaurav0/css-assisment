What is CSS ?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML Elements are to be display on screen, paper,or in other media.
  3. Css is not neccesaryCSS saves a lot of work.It can Control the layout of multiple web pages all at once.
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML document in 3 ways :

  • Inline - by using the stle attribute inside HTML elements.
  • Internal - by using a stle element in the head section.
  • External - by using a link element to link an external CSS file.
  • The most common way to add CSS , is to keep the stle external CSS files. However, in this tutorial we will use inline and internal styles,because this is easy to demonstrate , and easier for you to try it yourself.

When comparing CSS class vc ID , the difference is that CSS class applies a style to multiple elements, ID on the other hand , applies a style to one unique element . ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS , selectors are used to target a specific element or range of elemet on a web page . once an element has been targeted, a style or a set of styles cam be applied to the element.

There is wide range of selectors available . Two of the most common used are class and ID. Both are used to target elements to which a style should be applied.
